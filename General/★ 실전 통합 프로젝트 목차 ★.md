# ★ 실전 통합 프로젝트 목차 ★

## A. 기본 응용 (Level 1)

- 블루투스 체중계 (BLE + OLED + HX711)
- 온습도 IoT 로거 (DHT22 + Wi-Fi + MQTT)
- PIR 모션 감지등 (PIR + Relay + LDR)
- 조도 자동가로등 (BH1750 + PWM + MOSFET)
- 실시간 온도 표시기 (DS18B20 + LCD1602)
- 디지털 온도계 + EEPROM 데이터 저장
- 블루투스 시리얼 터미널 (HM-10 + UART)
- 초음파 거리계 (HC-SR04 + OLED)
- 버튼 인터럽트 기반 LED 메뉴 컨트롤러
- PWM 서보 제어기 (Timer + SG90 Servo)

- 환경 데이터 로거 (BME280 + RTC + SD)
- 가속도/진동 데이터 로거 (MPU6050 + SD)
- 실시간 그래프 전송기 (UART + CSV PC 출력)
- 전력 소비 측정기 (INA219 + OLED + SD)
- ADC 샘플링 오실로스코프 (ADC + DMA + TFT)
- Circular Buffer 기반 고속 센서 로거

- BLE + Wi-Fi 듀얼 프로토콜 센서 브릿지
